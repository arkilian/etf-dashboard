# ETF Dashboard

Dashboard em Streamlit para análise de ETFs e respetivas holdings.

## Stack
- Python
- Streamlit
- yfinance
- pandas

## Objetivo
Permitir carregar ETFs diferentes, enriquecer holdings com dados de mercado e visualizar métricas num dashboard.

## Instalar e usar
python -m venv .venv   # ou: python3 -m venv .venv
source .venv/Scripts/activate
python -m pip install --upgrade pip
pip install -r requirements.txt
